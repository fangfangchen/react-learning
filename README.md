## react-music-player

1. 参考来源: [慕课网](http://www.imooc.com/learn/868)

2. 功能说明：

	- 支持播放音乐
	- 支持前、后切换歌曲
	- 支持单曲循环、列表循环、随机播放
	- 支持调整播放进度
	- 支持调整音量
	- 支持查看我的歌曲列表
	- 支持删除歌曲

3. 查看效果

	执行命令： `node server`

	访问： `localhost:3000`
